# 🚀 TypeRacer Web Application

## 📌 Project Description

TypeRacer is a web-based typing speed game built using Angular.
The application allows users to test and improve their typing speed by racing against time or other players.

## ⚙️ Features

* ⌨️ Real-time typing test
* 📊 Words Per Minute (WPM) calculation
* 🎯 Accuracy tracking
* ⏱️ Timer-based race mode
* 🏆 Leaderboard (optional)
* 🌐 Multiplayer mode (optional)



## 🧠 How It Works

* A random text is generated and displayed to the user
* User types the text in a given time
* System calculates:

  * WPM (Words Per Minute)
  * Accuracy
* Results are shown at the end of the race
